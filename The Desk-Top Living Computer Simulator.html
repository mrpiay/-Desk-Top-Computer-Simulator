📖 Quick Reference - Instruction Codes:

  • 0 X - Set teacher's value to data desk X value
  • 1 X - Add value from data desk X to teacher's value
  • 2 X - Subtract value from data desk X from teacher's value
  • 3 X - Set data desk X value to teacher's value
  • 4 X - Jump to instruction desk X
  • 5 X - Jump to instruction desk X if teacher's value = 0
  • 6 X - Jump to instruction desk X if teacher's value < 0
  • 7 - Set teacher's value to Input desk value
  • 8 - Set Output desk value to teacher's value
  • 9 - End program
